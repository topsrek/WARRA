# Excel Format

## English

Excel (.xlsx) files can be directly opened in Microsoft Excel or other spreadsheet applications.

### Loading in Python

```python
import pandas as pd
df = pd.read_excel('filename.xlsx')
```

### Loading in R

```r
library(readxl)
df <- read_excel('filename.xlsx')
```

### Required Libraries

Python: pandas, openpyxl

R: readxl
