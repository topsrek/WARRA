# Parquet Format

## English

Parquet is a columnar storage format that offers excellent compression and performance for large datasets.

### Loading in Python

```python
import pandas as pd
df = pd.read_parquet('filename.parquet')
```

### Loading in R

```r
library(arrow)
df <- read_parquet('filename.parquet')
```

### Required Libraries

Python: pandas, pyarrow

R: arrow

## Deutsch
